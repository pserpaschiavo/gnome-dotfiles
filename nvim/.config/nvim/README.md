# Neovim Gruvbox Rainbow 60% Setup

Este repositório traz uma configuração moderna, minimalista e visualmente integrada do Neovim, inspirada no tema Gruvbox Rainbow e otimizada para teclados 60%.

## Estrutura do Repositório
- **FIRSTSTEPS.md**: Guia para iniciantes, primeiros passos e navegação básica.
- **INSTRUCOES.md**: Atalhos intermediários/avançados, plugins e dicas para usuários experientes.
- **HARDTIME.md**: Propósito e funcionamento do plugin Hardtime.nvim.
- **README.md**: Este arquivo, visão geral do projeto.

## Principais Plugins
- **Lazy.nvim**: Gerenciador de plugins
- **Lualine**: Statusline customizada
- **Telescope**: Busca de arquivos/texto
- **Treesitter**: Realce de sintaxe
- **nvim-cmp**: Autocompletar
- **Mason**: Gerenciador de LSPs
- **Gitsigns**: Integração com Git
- **Oil**: Gerenciador de arquivos em modo texto
- **Hardtime**: Treinador de produtividade
- **Copilot**: IA do GitHub
- **Bufferline**: Barra de buffers com ícones
- **Toggleterm**: Terminal flutuante
- **Noice/Notify**: Notificações pop-up e linha de comando moderna

## Instalação
1. **Pré-requisitos:**
   - Neovim 0.8+
   - Fonte Nerd Font (ProFont Nerd Font recomendado)
   - Git
2. Clone este repositório em `~/.config/nvim`:
   ```sh
   git clone <repo-url> ~/.config/nvim
   ```
3. Abra o Neovim e rode `:Lazy` para instalar os plugins.
