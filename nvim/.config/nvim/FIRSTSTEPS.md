# Primeiros Passos com Neovim

Este guia é para quem está começando no Neovim com este repositório!

## Abrindo o Neovim
- Execute `nvim` no terminal para abrir o editor.
- O explorador de arquivos (Oil) será aberto automaticamente se nenhum arquivo for especificado.

## Navegação Básica
- Use as setas ou `hjkl` para mover o cursor.
- Para abrir arquivos, navegue com o Oil e pressione `<Enter>`.
- Para salvar: `<leader>w` (normalmente `<Space>w`).
- Para fechar: `<leader>q` (`<Space>q`).

## Edição Simples
- Para entrar no modo de edição, pressione `i`.
- Para sair do modo de edição, pressione `<Esc>`.

## Buffers e Abas
- Troque de arquivos abertos com `<leader>bn` (próximo) e `<leader>bp` (anterior).
- Abra uma nova aba com `<leader>to`.

## Terminal Integrado
- Abra o terminal flutuante com `<leader>tt`.

## Plugins Visuais
- Ícones Nerd Font em todo o ambiente.
- Barra de buffers (Bufferline) para alternar entre arquivos facilmente.

## Recomendações
- Use sempre o leader (`<Space>`) antes dos atalhos customizados.
- Para instalar/atualizar plugins, use `:Lazy`.

Para instruções avançadas, consulte o arquivo [INSTRUCOES.md](INSTRUCOES.md).
